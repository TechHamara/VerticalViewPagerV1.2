# VerticalViewPager
Extension to create Vertical View Pager (For MIT AI2 and its distros)
### 1.Introduction 
**Description:** Using this extension you can create a vertical version of View Pager <br>
**Latest Version:** 1 <br>
**Last Updated:** 10-06-2021<br>
**Released:** 10-06-2021<br>
